Employee Performance Evaluation

This module allows you to log employee Evaluation based on your requirements.

Features

- Detail Employee Evaluation based on Evaluation Templates and Performance Period
- Easy Configuration and Easy to give Feedback and evaluation for manager and employees based on Inter and Intra Department
- Email Notifications to Specific configured Users / Evaluator / Reminder Email for Evaluator
- Evaluation based on Section wise and Full, can be configured in Evaluation Template



Installation

- Copy module to addons folder

- Install the module normally like other modules
